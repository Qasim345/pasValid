PasValid


pasValid is a jQuery plugin for password validation, used in signup form or atc.



Get Started

Installation and used

CDN link:

<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>
<script src="https://github.com/Qasim345/pasValid/js/pasValid.min.js"></script>

HTML code:

<div class="myinput pasValidInput">
    <input type="password" id="pass" />
</div>

JavaScript code:

<script type="text/javascript">
     $("#pass").pasValid();
</script>

Full example:

<form action="action.php" method="post">
    <h1>Sign up</h1>
    <input type="text" name="username" placeholder="Username" />
    <input type="email" name="email" placeholder="Email address"/>
    <div class="pasValidInput">
        <input type="password" id="pass"  name="password" placeholder="Password"/>
    </div>
    <button type="submit" name="submit">Sign up</button>
</form>

<script src="js/jquery.min.js"></script>
<script src="js/pasValid.min.js"></script>
<script>
    $("#pass").pasValid();
</script>

Output:

Sign Up
